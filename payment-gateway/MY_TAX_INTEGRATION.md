# Интеграция с API "Мой налог"

## Обзор

Интеграция с API "Мой налог" позволяет автоматически создавать чеки самозанятого и отправлять их в ФНС.

## Важные замечания

⚠️ **Официальный API "Мой налог" может быть ограниченным или недоступным.**

Возможные варианты интеграции:

1. **Официальный API** (если доступен)
   - Требует API ключ от ФНС
   - Может требовать OAuth авторизацию
   - Проверьте актуальную документацию на сайте ФНС

2. **SDK от ФНС**
   - Официальный SDK для работы с "Мой налог"
   - Может быть доступен на сайте ФНС

3. **Веб-интерфейс** (альтернатива)
   - Использование Selenium/Playwright для автоматизации
   - Вход через приложение "Мой налог"
   - Автоматическое создание чеков через веб-интерфейс

4. **Ручное создание** (fallback)
   - Чек создается локально
   - Данные сохраняются в БД
   - Позже синхронизируется через приложение или API

## Текущая реализация

### MyTaxService

Сервис `MyTaxService` реализует базовую интеграцию с API "Мой налог":

- `create_receipt()` - создание чека через API
- `get_receipt()` - получение информации о чеке
- `check_income_limit()` - проверка лимита дохода
- `sync_receipt_to_my_tax()` - синхронизация локального чека

### Fallback механизм

Если API недоступен:
1. Чек создается локально с временным номером
2. Данные сохраняются в БД
3. Позже можно синхронизировать через метод `sync_receipt_to_my_tax()`

```env
# API ключ (если доступен)
MY_TAX_API_KEY=your_api_key_here
```

### Получение API ключа

1. Зарегистрируйтесь в приложении "Мой налог"
2. Проверьте наличие API в настройках
3. Если API недоступен, используйте альтернативные методы

## Использование

### Создание чека

```python
from app.services.my_tax_service import MyTaxService

my_tax = MyTaxService()

receipt_data = await my_tax.create_receipt(
    amount=Decimal("5000.00"),
    client_name="ООО \"Рога и Копыта\"",
    client_type="legal_entity",
    client_inn="1234567890",
    service_name="Услуги по подбору персонала",
    email="accounting@example.com"
)
```

### Проверка лимита дохода

```python
income_status = await my_tax.check_income_limit(year=2024)

if income_status:
    print(f"Доход за год: {income_status['total_income']}")
    print(f"Осталось до лимита: {income_status['remaining']}")
    print(f"Лимит превышен: {income_status['limit_exceeded']}")
```

## Альтернативные методы

### Вариант 1: Интеграция через веб-интерфейс

Если API недоступен, можно использовать автоматизацию браузера:

```python
from playwright.async_api import async_playwright

async def create_receipt_via_web(receipt_data):
    async with async_playwright() as p:
        browser = await p.chromium.launch()
        page = await browser.new_page()
        
        # Вход в "Мой налог"
        await page.goto("https://lknpd.nalog.ru")
        # ... авторизация и создание чека
        
        await browser.close()
```

### Вариант 2: Ручное создание через приложение

1. Экспорт данных о платежах из БД
2. Ручное создание чеков в приложении "Мой налог"
3. Импорт номеров чеков обратно в БД

## Troubleshooting

### Ошибка: "API endpoint not found"
- API может быть недоступен
- Используйте локальное создание чеков

### Ошибка: "Unauthorized"
- Проверьте правильность API ключа
- Проверьте срок действия токена

### Ошибка: "Connection timeout"
- Проверьте доступность API
- Используйте fallback механизм
